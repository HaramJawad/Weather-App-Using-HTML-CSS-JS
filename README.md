A weather app built with HTML, CSS, and JavaScript. It shows the current weather conditions, daily forecast, and hourly forecast for any city using a weather API.
FEATURES: 
Search weather by city name, Shows current temperature, humidity, wind, precipitation, and “feels like” temperature, Daily forecast for the next 3 days, Hourly forecast for each day (first 8 hours), Unit conversion for temperature (°C/°F), wind (kph/mph), and precipitation (mm/in), Responsive layout, Handles errors like “city not found” or API issues. 

HOW TO USE:
Enter the name of a city in the search bar. Click Search. See the current weather and forecast. Change units from the dropdown if needed.

TECH STACK: 
HTML5 for structure, CSS3 for styling, JavaScript for fetching data from WeatherAPI and updating UI dynamically.

API USED:
WeatherAPI – Free plan for getting weather data. 

NOTE: Replace API_KEY in script.js with your own WeatherAPI key to run this project.
